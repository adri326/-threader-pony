# pony-threader

**This is a PoC for an [issue](https://github.com/ponylang/ponyc/issues/3013) of [Pony](https://ponylang.io/)!**

## Installation & Usage

```sh
# clone
git clone https://github.com/adri326/-threader-pony threader-pony
cd pony-threader

# setup
mkdir build
cd build
cmake ..

# build
make

# run
./threader-pony
```
